# Basic API Example

This example demonstrates
- Default router
- GET routes
- JSON responses
- URL params
- Basic routing

## Run

Two ways of running:

**1. Vodka CLI(recommended)**

Use the vodka CLI to enable hot reload on file changes

```bash
vodka
```

If not installed, you can install it using

```bash
go install github.com/DevanshuTripathi/vodka/cmd/vodka@latest
```

**2. Using Go directly**

```bash
go run .
```
